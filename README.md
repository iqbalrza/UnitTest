# ✅ Unit Test — Pengujian Perangkat Lunak dengan Python

Implementasi **unit testing** menggunakan modul `unittest` bawaan Python untuk memastikan kebenaran fungsi dan logika program.

## 📌 Deskripsi

Repo ini berisi contoh penerapan unit test di Python sebagai praktik pengembangan perangkat lunak yang baik (clean code & TDD — Test-Driven Development).

## 🛠️ Teknologi

![Python](https://img.shields.io/badge/Python-3776AB?style=flat&logo=python&logoColor=white)

**Library:**
- `unittest` — Framework testing bawaan Python

## 📁 Struktur File

```
└── import unittest.py    # Script unit test
```

## 📚 Konsep yang Dibahas

- `TestCase` — Membuat kelas pengujian
- `assertEqual`, `assertTrue`, `assertRaises` — Assertion methods
- `setUp` & `tearDown` — Persiapan dan pembersihan test
- Menjalankan test suite
- Menafsirkan hasil test (pass/fail/error)

## 🚀 Cara Menjalankan

1. Clone repo ini:
   ```bash
   git clone https://github.com/iqbalreza/UnitTest.git
   ```
2. Jalankan test:
   ```bash
   python "import unittest.py"
   ```

   Atau dengan verbose output:
   ```bash
   python -m unittest "import unittest.py" -v
   ```
